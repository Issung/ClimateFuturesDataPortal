Introduction

Climate Futures is basically a data portal which is used for providing climate related data to anyone based upon request. This very idea was initiated by The Climate Future Team to bring a faster data processing system. This web-based application contains a front-end that would allow you to discover what data is available and extract what you desire.

In this application, you can extract data subset with the following attributes:

  • Variables: humidity, temperature, wind direction, etc. There are up to 100 variables.
  • Spatial extent: single location, a bounding box, or a defined polygon (geospatial file).
  • Time period: single time slice, or a period of time from start to end that is defined by you.
  • Time resolution helps to sort the real time data which might be added every hour,day or month.

Futhermore, There is a search option if you wish to filter particular variable. After all the selection is done a new page will showcase a summary of all the selections you made.
