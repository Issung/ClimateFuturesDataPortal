Introduction

The Climate Futures data portal allows for easy access to 150Tb of climate data based on filters set by the user.

The climate database and data portal system is maintained by the Climate Futures team.

In this application, climate data may be searched and downloaded by setting the following filters:

  • Spatial extent: A rectangular geospatial area of which to search for climate data.
  • Time period: A period of time defined by 2 dates, to search within.
  • Variables: Select variables desired for extraction (e.g humidity, temperature, etc.). There are currently 59 variables.
  • Time resolution: helps to sort the real time data which might be added every hour, day or month.

This user manual outlines the different features & functions of the data portal and how to use them, incase you require any assistance.

Different pages of the user manual can be accessed to the left, and you may return to the data portal using the button in the bottom left of the screen.
