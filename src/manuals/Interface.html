Knowing the user interface

Introduction of the interface of the application.

  • Area selection

    • This area contains the section where you can proceed your location selection for data extraction. It contains a map API from ArcGIS and functionality the team implements to help you choose area of your choice, either by square or importing a shapefile.

In this application, you can extract data subset with the following attributes:

  • Variables: humidity, temperature, wind direction, etc. There are up to 100 variables.
  • Spatial extent: single location, a bounding box, or a defined polygon (geospatial file).
  • Time period: single time slice, or a period of time from start to end that is defined by you.
  • Ensemble part: all models or only a subset.

Futhermore, the system will inform you of the approximate data request size prior to initiating extraction.
